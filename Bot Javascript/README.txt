Para iniciar usamos:
npm init -y

Depois devemos trazer o recurso express para o projeto,então devemos instalá-lo:
npm install express 

Para rodar o servidor que está sendo criado usa-se:
node ./"NOME ARQUIVO JAVASCRIPT"
Devemos instalar o recurso socket para funcionamento de nosso servidor usando o seguinte comando:
npm install socket.io 
Para interromper qualquer processo usa-se:
CTRL +C
Para fazer o servidor verificar cada atualização podemos fazer com que o mesmo assista usando:
node --watch ./"NOME ARQUIVO JAVASCRIPT"
